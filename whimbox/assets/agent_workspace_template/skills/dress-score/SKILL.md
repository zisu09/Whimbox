# Dress Score

当用户要求你对穿搭进行打分点评时，使用本 skill。

## 工作流

1. 调用`nikki.goto_ui`工具，前往换装界面
2. 判断工具调用是否成功。如果失败，停止工作流并告知用户；如果成功，继续工作流
3. 调用 `analyze_image`，使用 `mode="screenshot"`
4. 工具返回结果后，再用自然语言整理给用户。

## 调用`analyze_image`工具时直接使用如下prompt

- **忽略画面中的UI元素**（如按钮、文字、边框、任务提示等所有非角色内容）
- **只关注画面中的角色装扮**
- 重点评价角色的发型、妆容、饰品、服饰、整体协调性、创意性等

**评分维度**：

- 发型：造型、适配度等（满分10分）
- 妆容：妆容风格、色彩搭配、适配度等（满分10分）
- 饰品：各类饰品的搭配效果、适配度等（满分10分）
- 服饰：服装款式、色彩、质感等（满分10分）
- 整体协调性：全套装扮的整体搭配和谐度（满分10分）
- 创意性：装扮的独特性和个人风格（满分10分）

**输出格式**：

- 总分（0-10，只保留一位小数）
- 分项分数和一句话点评（闪光配件无则不展示）
- 一句话总结最大优点
- 0-2条改进建议
- 总体字数控制在150字以内
